## Getting Started

### Prerequisites

- Flutter SDK (^3.7.2)
- Dart SDK (^3.7.2)
- Android Studio / VS Code
- Git

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/reqres_app.git
```

2. Navigate to project directory
```bash
cd reqres_app
```

3. Install dependencies
```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```
